Open Redirect Vulnerability Demonstration

This lab demonstrates an Open Redirect vulnerability. You can input any URL and be redirected to it.

Redirect to Another URL

Instructions:

1. Enter a URL (e.g., https://google.com or https://example.com) in the input field.

2. Click the "Redirect" button and observe that you will be redirected to the URL you entered.

3. The application does not validate or sanitize the input URL, allowing redirection to any site, including potentially malicious ones.
