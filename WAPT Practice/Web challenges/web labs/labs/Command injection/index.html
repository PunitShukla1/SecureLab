Command Injection Vulnerability Lab

This lab demonstrates how a command injection vulnerability can allow an attacker to execute arbitrary JavaScript commands.

Lab Instructions

In this lab, you will input a command into the form below. If the application is vulnerable, the command will be executed in the browser. This simulates how command injection flaws could occur.
Warning: Only commands that do not affect your system should be tested in this lab.

Output

Example Injected Commands

  • alert('Injected!');
  • console.log('Command Executed');
  • document.body.style.backgroundColor = 'red';
